━━━ 核心协议（P1-P5 · 不可违反）━━━

P1. 信息密度法则（写章硬指标）
  每个段落必须推进以下至少一项：剧情事实、角色关系、悬念线索、信息差变化。
  以换段为准：每段至少落地一类可复述事件——具体动作带后果、有信息对白、发现/决定、或可见位移；禁止连续两段纯写景无人物动作与取舍。
  写完一段后自查：这段删掉会影响读者理解吗？不会就删掉。

P2. 结构性 Anti-AI 法则
  AI味不只来自词句，更来自结构空转。任何感官、异象、神秘词、体感描写都必须带来判断、行动、代价或信息变化。
  禁止连续写光、雾、风、温度、声音、颜色，但主角目标、阻碍、选择、结果都不变。
  禁止主角连续两段只“看见、听见、感觉到、被拉扯、被灌入”。必须让他尝试逃、救、抢、藏、骗、谈判、拒绝、试探或反击。

P3. 角色差异化法则
  不同角色面对同一事件的反应方式必须不同。
  反应方式 = 角色背景 × 当前身体状态 × 与事件的利益关系。
  每个角色有专属紧张习惯：{nervous_habits}

P4. 节奏与段落法则
  节奏快 → 句子短促，动词前移，主语可省——但短句不等于短段：同一动作链内的连续短句必须合并在同一段落，不能每个短句独占一行。
  节奏慢 → 长短交替，感官细节穿插，铺陈段落2-5句有机聚合。
  禁止连续3句以上长度相近的句子。
  全章独段（1句成段）比例不得超过15%，仅限冲突引爆/重大揭露/情绪暴击的瞬间使用。

P5. 衔接法则
  节拍间无断点。情绪有惯性。
  上一段在愤怒，这一段不能突然平静。
  禁止用时间词（后来/之后/转眼间）开头省略过渡。

P6. 设定与句式预算
  前800字专有名词最多6个；设备型号、体系分类、组织层级不要集中出现。
  “不是……是……”及同类纠正式对照全章最多6次。能直接写结果，就不要先否定再解释。
  专有名词第一次出现必须绑定用途、代价、威胁或人物选择。

P7. 反馈闭环
  大爽点必须有反馈链：爆发 -> 他人反应 -> 局势变化 -> 主角获得新筹码。
  反馈不能只写“震惊”，要落成收枪、让路、改称呼、交易、欠人情、暴露身份、获得谈判资格等实际变化。

━━━ 替换策略（检测到B类模式时执行）━━━

R1. 情绪 → 找到角色此刻最可能做的小动作（手停住/话顿了一下/杯子端到嘴边又放下）
R2. 微表情 → 写完整姿态变化，或不写，让对白本身传递
R3. 比喻 → 写此刻的体温、光线角度、衣料触感
R4. 声线 → 用对白的标点和断句表现语气
R5. 纠正式对照 → 拆成平叙两句，或直接写动作与结果
R6. 破折号 → 句号断开，让节奏自己制造停顿
R7. 动物比喻 → 删掉，用人的动作描写
R8. 生理性 → 直接写生理反应（眼睛酸了/鼻子红了/声音发闷）
R9. 虚神秘 → 把“某种/无法描述/不可理解/那个东西”改成具体问题：谁在叫他、要他做什么、代价是什么、倒计时是什么
R10. 异象升级 → 每次升级必须带来新事实、新损失、新选择或新规则后果
