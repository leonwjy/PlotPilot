「本章你要讲的这段故事」

{outline}

━━ 写的时候记住 ━━

• 每个换段（空行分段）里至少要有：具体动作、有信息的对话、发现/决定、或可见位移之一；不要连续两段纯写景凑字。
• 别平均使力。冲突爆发的地方多写两百字也值得，过渡的地方一笔带过就行。
• 场景里的人不能是纸片人。哪怕配角，也要有一个小动作或者眼神，让他活过来。
• 结尾别收干净。留一根刺——一个问题没回答、一个人转身走了、一句话说到一半停了。让读者非看下一章不可。
• 每个场景都要'双线并行'——表面事件 + 底层暗流。读者看完后应该觉得'这件事不简单'。
• 如果上一章结束时角色处于某种状态（紧张/受伤/震惊），本章开头必须先接住这个状态再展开新情节。前章的悬念钩子、情感余韵、场景状态在本章前三句话内必须有回响。
• 情绪通过动作和感官细节传递——不写'他感到愤怒'，写'他端起杯子又放下了'。
• ⚠️ 段落聚合铁律：属于同一视觉焦点、同一动作链或同一心理转折的句子必须合并在同一段落里。不要把连贯的动作拆成一行一句的分镜脚本——"她走到窗前。她停下了。她看着窗外。" 应合并为 "她走到窗前停下，目光落在窗外。" 只有冲突引爆/重大揭露/情绪暴击的瞬间，才用一句独段制造暴击感，但全章独段比例不得超过15%。
• 本章必须有一个可复述的故事单元：目标、阻碍、行动、兑现、收获/代价、新期待。写完后读者应该能说出“主角这一章想做什么、被什么卡住、最后局势怎么变了”。
• 如果大纲没有写清困境，就在不改主线的前提下补出期限、后果、误判、资源不足或关系压力；不要写成“主角走路、看景、想起过去”的空转章。
• 每个重要场景都要有情绪缺口：读者要么想看主角脱困，要么想看反击，要么想看秘密揭开，要么想看关系变化。缺口不能靠形容词堆出来。
• 爽点必须有反馈。赢了以后写清楚获得了什么、谁的态度变了、信息差怎么变了、下一步主动权在哪里。
• 形容词和比喻要省着用。同一段里不要连续堆颜色、气味、质感、抽象比喻；景物必须牵动人物选择、危险或信息差。
• 前800字专有名词最多6个。先让读者记住主角、目标、倒计时、阻碍，体系名、设备型号、组织细分延后。
• “不是……是……”结构全章最多6次，优先直接写动作、事实和后果。
• 每个大爽点都要写出反馈链：爆发后，别人具体怎么改变态度/动作，局势给主角什么新筹码。

{beat_section}

讲吧。
