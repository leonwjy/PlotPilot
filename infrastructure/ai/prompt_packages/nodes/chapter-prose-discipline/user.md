━━━ 行文戒律（反八股 / 控水分）━━━

【本质】
正文不是说明文，也不是情绪词堆叠。每一段都要让故事往前走。

【故事单元】
- 本章必须围绕一个可复述的故事单元推进：目标、阻碍、行动、转折、兑现/代价、新期待。
- 如果大纲只写“发现异常、前往某地、想起往事”，正文必须在不改核心情节的前提下补出行动理由和失败后果。

【信息密度 / 情节密度】
- 大约每 300～500 字至少出现一条可复述的新事实：人物立场、线索、因果、决策、关系变化或新风险。
- 以空行分段为准，每段至少包含一项：具体动作并带轻微后果、有信息量的对白、发现/误会/决定、可见位移或关系试探。
- 禁止连续两段只有景物、气味、光色、体感或抽象氛围。

【去 AI 味】
- 纠正式对照（全章禁止高频）：禁止使用「不是……而是……」「从来不是……」「真正的……」这类解释腔来替代事件推进。
- 破折号：不要用破折号拖尾补充、叠比喻或把同一意思换说法强调；需要停顿时，用句号和动作承接。
- 禁止用形容词、抽象比喻、重复震惊、哲学总结凑字。
- 禁止用「某种说不清的东西」「有什么东西变了」「一切才刚刚开始」替代具体事件。
- 情绪要通过动作、对白、停顿、选择和后果传递，不要直接贴“愤怒、悲伤、震惊、复杂”等标签。

【扩写方法】
- 字数不够时，增加动作链、对白博弈、误判、资源代价、旁人反馈、局势变化；字数已接近目标时，必须删去重复反应、重复体感、重复环境铺陈。
- 冲突爆发处可以慢写；过渡处一笔带过。
- 爽点必须有反馈链：爆发 -> 他人反应 -> 局势变化 -> 主角获得新筹码。

【字数边界】
- beat_target_words 是目标，不是最低消费额度。正文应落在目标附近，允许略低，不允许为了“写透”大幅超出。
- 如果 partition_mode 为 single，当前 beat 就是整章正文：超过目标后只补必要的兑现和钩子，禁止继续增加新回合、新旁观者反应或新的解释段。
- 如果已经写完目标、阻碍、行动、转折、兑现/代价、新期待六项，即使还能展开，也必须收束。

【段落】
- 段落是意义集合，不是分镜脚本。属于同一视觉焦点、同一动作链或同一心理转折的句子要合并。
- 一句独段只用于冲突引爆、重大揭露或情绪暴击，全章比例要少。
